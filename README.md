## 项目简介

这是一份**从零基础出发**的易经系统学习笔记，采用「对话讨论 + 课后练习」的模式逐步推进。

笔记内容涵盖《易经》的基础框架（阴阳、五行、八卦、六十四卦）、起卦方法、解卦逻辑，最终目标是能够**独立起卦解卦，辅助日常决策**。

笔记风格力求**通俗易懂**——用生活化的比喻解释抽象概念，配合思考练习和自测题巩固理解。

## 适用人群与使用说明

### 适合谁

| 类型 | 说明 |
|:---|:---|
| 零基础初学者 | 对阴阳有模糊概念，但五行、八卦不了解 |
| 有古文基础的学习者 | 能阅读文言文，希望系统理解《易经》结构 |
| 想学起卦解卦的人 | 最终目标是能实际用于决策参考 |

### 如何使用这套笔记

1. **按顺序阅读**：笔记按阶段编排，建议从第一阶段开始，循序渐进
2. **做练习题**：每课后附有思考题和选择题，先独立思考再看答案
3. **配合学习手册**：用 `I-Ching_00_学习手册.md` 追踪自己的学习进度
4. **每天一小时**：笔记设计为每天投入约 1 小时即可稳步推进

## 核心知识点速览

### 阴阳

> 万物变化的根本——描述任何事物内部两种**相反又相互依存**的力量。

- **互根**：没有阴就没有阳，必须同时存在才能互相定义
- **消长**：阴阳此消彼长，像跷跷板（如一天之中阳气从清晨到正午渐盛，午后渐消）
- **转化**：阴到极点转为阳，阳到极点转为阴（"否极泰来"）

### 五行

> 不是五种物质，而是五种**运动状态/变化阶段**。

| 五行 | 代表状态 | 相生 | 相克 |
|:---:|:---|:---|:---|
| 木 | 生发、向上、舒展 | 生火 | 克土 |
| 火 | 炎热、上升、极盛 | 生土 | 克金 |
| 土 | 稳定、转化、承载 | 生金 | 克水 |
| 金 | 收敛、下降、肃杀 | 生水 | 克木 |
| 水 | 寒冷、向下、潜藏 | 生木 | 克火 |

**速记口诀：** 相生「木火土金水，挨着就是生」；相克「木土水火金，隔一个就是克」

### 八卦

> 三个爻叠成一卦，共八卦，代表天地间八种基本力量。

| 卦名 | 象征 | 五行 | 错卦 |
|:---:|:---:|:---:|:---:|
| 乾 ☰ | 天 | 金 | 坤 ☷ |
| 坤 ☷ | 地 | 土 | 乾 ☰ |
| 震 ☳ | 雷 | 木 | 巽 ☴ |
| 巽 ☴ | 风 | 木 | 震 ☳ |
| 坎 ☵ | 水 | 水 | 离 ☲ |
| 离 ☲ | 火 | 火 | 坎 ☵ |
| 艮 ☶ | 山 | 土 | 兑 ☱ |
| 兑 ☱ | 泽 | 金 | 艮 ☶ |

### 六十四卦

> 两个八卦叠加（8×8=64），每卦六爻，从下往上：初、二、三、四、五、上。

**经典案例——泰卦与否卦：**

- **泰卦**（上坤下乾）：天地相向运动，相遇相通 → 通泰亨通
- **否卦**（上乾下坤）：天地背向运动，越走越远 → 闭塞困顿
- **核心智慧**：表面"颠倒"反而通畅，表面"正常"反而闭塞
