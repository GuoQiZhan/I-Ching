## 第一课：阴阳

### 什么是阴阳

你可能听说过：阴 = 黑、冷；阳 = 亮、暖。这些都没错，但只是**举例**，不是**本质**。

> **核心定义：** 阴阳是一种**关系模型**——描述任何事物内部两种**相反又相互依存**的力量。

举个例子：

- 没有"冷"，你根本不知道什么叫"暖"——它们必须**同时存在**才能互相定义
- 白天是阳，夜晚是阴——但白天里有最热的中午（**阳中之阳**：一天中阳气最盛的时刻）和渐凉的下午（阳中之阴）
- 你呼气是阳（向外），吸气是阴（向内）

### 阴阳的三个关键性质

**1. 阴阳互根**

没有阴就没有阳，没有阳就没有阴。它们不能独立存在。就像没有"上"就没有"下"，没有"难"就没有"易"。

**2. 阴阳消长**

阴阳不是静止的，它们此消彼长，像跷跷板。

> 一天之中：清晨阳气渐升 → 正午阳极 → 下午阴气渐生 → 午夜阴极 → 又开始循环。

**3. 阴阳转化**

阴到极点会转为阳，阳到极点会转为阴。

> "物极必反""否极泰来"——这就是阴阳转化。《易经》64卦里的"泰卦"和"否卦"就是这个意思。

### 思考练习

**Q1：** 一杯热水放在桌上变凉，这个过程中"阳"和"阴"是怎么变化的？

**Q2：** 一个人从健康到生病再到康复，用阴阳的角度怎么看？

<details>
<summary>▸ 点击查看参考答案</summary>

**A1：** 热水一开始是极阳状态，随后阳气渐消、阴气渐生，逐渐变凉，最终趋于室温——即阴阳达到平衡状态。

**A2：** 健康是**阴阳平衡**的状态。生病是平衡被打破——可能是阴太盛（如怕冷、腹泻）或阳太盛（如发烧、上火）。康复就是重新恢复平衡的过程。

</details>

## 第二课：五行

### 五行的本质

五行**不是五种物质**，而是**五种运动状态 / 变化阶段**。

"行"这个字的意思是"行走、运行"——所以五行是描述事物**变化过程中五个阶段的循环**：

| 五行 | 不是指这个物质 | 而是指这种"状态" |
|:----:|:----------:|:------------:|
| **木** | 不是木头 | 生发、向上、舒展 |
| **火** | 不是火焰 | 炎热、上升、极盛 |
| **土** | 不是泥土 | 稳定、转化、承载 |
| **金** | 不是金属 | 收敛、下降、肃杀 |
| **水** | 不是河水 | 寒冷、向下、潜藏 |

> 木和火偏**阳**（上升、外放），金和水偏**阴**（下降、收敛），**土**居中央起调和作用。

### 相生 —— "妈妈生孩子"

相生是**促进、滋养**的关系，形成一个循环：

> **木 → 火 → 土 → 金 → 水 → 木（循环）**

| 相生关系 | 理解方式 |
|:------:|:------:|
| **木 生 火** | 钻木取火，木头燃烧生火 |
| **火 生 土** | 火烧完留下灰烬，化为土 |
| **土 生 金** | 金属矿物蕴藏于大地之中 |
| **金 生 水** | 金属表面遇冷凝结水珠；金性收敛凝聚，极则化液 |
| **水 生 木** | 水滋养草木生长 |

### 相克 —— "克制、制约"

相克是**抑制、约束**的关系，也形成一个循环：

> **木 → 土 → 水 → 火 → 金 → 木（循环）**

| 相克关系 | 理解方式 |
|:------:|:------:|
| **木 克 土** | 树木的根扎进土里，把土"破"了 |
| **土 克 水** | 土能挡水——"兵来将挡，水来土掩" |
| **水 克 火** | 水可以浇灭火焰 |
| **火 克 金** | 火可以融化金属 |
| **金 克 木** | 金属可以锯断木头 |

### 速记口诀

```
相生：木火土金水，挨着就是生
相克：木土水火金，隔一个就是克
```

对照验证：木隔一个是土（木克土），土隔一个是水（土克水）……依次验证，全部成立。

## 第三课：八卦

### 爻 ——《易经》的最小单位

《易经》最基本的符号叫**爻**（读 yáo），只有两种：

| | 阳爻 | 阴爻 |
|:--:|:---:|:---:|
| 符号 | `━━━━━━` | `━　━` |
| 形态 | 一条完整横线 | 一条中间断开的横线 |
| 代表 | 刚健、积极、主动 | 柔顺、被动、承载 |

**三个爻叠在一起**，就组成一个**卦**，叫**八卦**。

为什么是三个？因为古人认为**天地人三才**构成世界，三爻分别代表天（上）、人（中）、地（下）。

### 八卦总览

| 卦名 | 符号 | 象征 | 五行 | 性质 |
|:---:|:---:|:---:|:---:|:----:|
| **乾** | ☰ | 天 | 金 | 刚健、创造 |
| **坤** | ☷ | 地 | 土 | 柔顺、承载 |
| **震** | ☳ | 雷 | 木 | 震动、奋起 |
| **巽** | ☴ | 风 | 木 | 渗入、顺从 |
| **坎** | ☵ | 水 | 水 | 险陷、流动 |
| **离** | ☲ | 火 | 火 | 光明、依附 |
| **艮** | ☶ | 山 | 土 | 停止、稳定 |
| **兑** | ☱ | 泽 | 金 | 喜悦、开口 |

### 八卦的阴阳配对（错卦）

八卦是**两两配对**的，每对阴阳完全相反，每一爻都互为镜像。这种配对叫**"错卦"**或**"旁通卦"**：

| 配对 | 卦 | 符号 | 象征 | 说明 |
|:---:|:---:|:---:|:---:|:---:|
| **第一对** | 乾 | ☰ | 天 | 三个阳爻，纯阳之极 |
|  | 坤 | ☷ | 地 | 三个阴爻，纯阴之极 |
| **第二对** | 坎 | ☵ | 水（险） | 阳被阴包围 |
|  | 离 | ☲ | 火（明） | 阴被阳包围 |
| **第三对** | 震 | ☳ | 雷（动） | 阳气从底部升起 |
|  | 巽 | ☴ | 风（入） | 阴气从底部渗入 |
| **第四对** | 艮 | ☶ | 山（止） | 阳气到顶，止住不动 |
|  | 兑 | ☱ | 泽（悦） | 阴气到顶，开口外泄 |

> **核心思维：** 《易经》里没有任何一卦是独立存在的，每一卦都活在一个关系网里。单独看一卦只能看到一面，配对看才能看到全貌。

### 三爻与天地人

三爻的位置是固定的：

| 位置 | 对应 | 含义 |
|:---:|:---:|:---:|
| 最上面一爻 | **天** | 大势、超越的力量（你控制不了的） |
| 中间一爻 | **人** | 你的选择和行动（你能控制的） |
| 最下面一爻 | **地** | 环境和基础条件（你部分能影响的） |

> 解卦时看**哪一爻动了**，就知道问题出在哪个层面——是根基问题、人事问题、还是大势问题。

## 第四课：六十四卦

### 64卦是怎么来的

三爻成一卦，共八卦（8个）。但《易经》有64卦，怎么来的？

> 答案：两个八卦叠在一起。
> **上面三爻 = 上卦（外卦）**
> **下面三爻 = 下卦（内卦）**
> 八 × 八 = 六十四

这样每一卦有**六个爻**，从下往上数为：

```
初 → 二 → 三 → 四 → 五 → 上
```

### 案例：泰卦与否卦

这是《易经》中最经典的对比——

| | 泰卦 ☷☰（上坤下乾） | 否卦 ☰☷（上乾下坤） |
|:--|:---:|:---:|
| **位置** | 地在上，天在下 | 天在上，地在下 |
| **运动** | 天性向上、地性向下，相向而行 | 天继续向上、地继续向下，越走越远 |
| **结果** | 相遇、相交、**相通** | 彼此脱离、**不交流** |
| **象征** | **通泰、亨通、平安** | **闭塞、不通、困顿** |
| **本质** | 表面"颠倒"，实际通畅 | 表面"正常"，实际停滞 |

> **核心智慧：** 表面"颠倒"的反而通畅，表面"正常"的反而闭塞。
> 成语**"否极泰来"**就从这来——困到极点就会转为通泰。

### 《易经》的文字结构

64卦不光是符号，每一卦都有文字解释：

- **卦辞**：解释这一卦整体代表什么
- **爻辞**：解释每一爻（六个位置）分别代表什么

这些文字合起来就是《易经》的**"经文"**，传说是**周文王**写的。

后世还有**"传"**（解释经文的文字），最有名的是**《十翼》**，传说是**孔子**写的。

### 开放思考题

**Q1：** 用你自己的话，向一个完全不懂的朋友解释《易经》是什么，你会怎么说？

**Q2：** 如果你面前是"上乾下坤"的卦，你认为这代表什么？为什么天在上地在下反而不好？

**Q3：** 五行中，"火克金"和"金生水"是否矛盾？

<details>
<summary>▸ 点击查看参考答案</summary>

**A1：** 《易经》用64种变化模式，描述天地人事物发展的规律，帮助人在变化中找到方向。它的基础是阴阳——万物都由两种相反相成的力量构成，通过观察这些力量的变化，可以推断事物的发展趋势。

**A2：** 这是否卦。天的性质向上，地的性质向下——两者越走越远，不再交流。表面看似"正常"和"安稳"，实际意味着停滞不前、闭塞不通。这告诉我们：表面正确的事物未必是好的，关键是是否有交流和互动。

**A3：** 不矛盾。五行的关系不是单一的"好"或"坏"，而是**多角度的关系网**。就像人与人之间：A可能是B的老师（A生B），但A同时也可能约束B的行为（A克B）。五行之间同时存在相生和相克的关系，这才构成了完整的动态平衡。

</details>

## 第五课：卦辞入门

### 卦辞是什么

你已经知道64卦由两个八卦叠加而成，每卦六爻。但光有符号，你不知道这一卦在"说什么"。

> **卦辞 = 挂在每一卦下面的一句话，告诉你这一卦整体代表什么处境。**

例如泰卦：

```
泰：小往大来，吉，亨。
```

六个字，把整个卦的核心意思说清楚了。

---

### 卦辞的两个零件

卦辞不是随意的感叹句，它有固定结构，一般包含两部分：

**① 叙述句 —— 描述处境**

用比喻或象征告诉你"现在是什么情况"：

| 卦 | 叙述句 | 意思 |
|:---:|:---|:---|
| 屯 | "勿用有攸往，利建侯。" | 创业初期，不要远行，利于建立根基 |
| 蒙 | "匪我求童蒙，童蒙求我。" | 不是我去找学生，是学生来找我 |
| 需 | "有孚，光亨，贞吉，利涉大川。" | 有信心等待，时机光明，可以渡河 |

叙述句像在讲一个场景，这是卦辞的灵魂。

**② 断语 —— 给出判断**

叙述之后，会跟一个"结论"，告诉你吉还是凶、该做还是不该做。断语用的是一套固定术语——

---

### 断语光谱：从好到坏

《易经》的断语不是简单的"好/坏"二分法，而是一个光谱：

```
吉 ──→ 吝 ──→ 悔 ──→ 厉 ──→ 凶
好        小问题    能改过    有危险    坏
```

| 断语 | 本质 | 类比 |
|:---:|:---|:---|
| **吉** | 顺利、得当、合道 | 天时地利人和，做对了 |
| **吝** | 有小问题、格局不够 | 方向没错，但做法不够好，有点可惜 |
| **悔** | 有过失，但能反省改过 | 做错了，但知道错在哪，可以回头 |
| **厉** | 有危险，但还没出事 | 走在悬崖边，小心则无事 |
| **凶** | 不顺、失当、违背规律 | 逆势而动，怎么做都不对 |

> **悔 vs 厉 的区别：** 悔 = 已经错了，正在回头（有知觉的错误）；厉 = 还没出事，但处境本身很险（无知觉的危险）。厉比悔更危险。

---

### 亨与贞：两个高频状态词

这两个字在卦辞里出现频率极高，但不是断语，而是**状态描述**：

**亨 = 亨通、通达**

> 描述事物发展没有阻塞。相当于"这件事的通道是打开的"。

**贞 = 正、坚守正道**

> 不是"结果好"，而是"你守住了正确的方向"。它是**行为建议**，不是结果预告。

所以 **"亨，利贞"** 的意思是：

> 通道是通的（亨），利于坚守正道（利贞）——事情能成，但前提是你别走歪路。

---

### 元亨利贞：四德

这四个字出现在很多卦的卦辞里，被称为"四德"：

| 字 | 含义 | 作用 |
|:---:|:---|:---|
| **元** | 大、始、根本 | 事物的起点和根基 |
| **亨** | 通达 | 事物发展没有阻塞 |
| **利** | 适宜、有利 | 某种行为适合去做 |
| **贞** | 正、坚守 | 坚守正道，不偏不倚 |

**核心逻辑链：**

> **元亨利贞 = 根基好 → 通道顺畅 → 做事有利 → 守正则吉**

乾卦的卦辞就是这四个字：

> **乾：元，亨，利，贞。**

> **重要提醒：** "利贞"不是"一定吉利"。"利贞"真正的意思是——"如果你守住正道，就有利。"它暗含一个条件，是行为指引，不是空头支票。

### 练习

**练习一：断语排序**

将以下断语从"最好"到"最差"排序：

> 凶 · 吝 · 吉 · 厉 · 悔

<details>
<summary>▸ 答案</summary>

**吉 > 吝 > 悔 > 厉 > 凶**

- 吉：顺利合道
- 吝：小问题，格局不够
- 悔：做错了，但能反省
- 厉：处境危险，还没出事
- 凶：逆势而动，怎么做都不对

</details>

---

**练习二：古文直译**

翻译以下三句卦辞，说出你的理解：

> **① 屯：元亨，利贞。勿用有攸往，利建侯。**
>
> **② 蒙：亨。匪我求童蒙，童蒙求我。**
>
> **③ 需：有孚，光亨，贞吉，利涉大川。**

<details>
<summary>▸ 参考答案</summary>

**① 屯卦：**

> 基础亨通，利于守正。不要主动远行，利于扎下根基、建立根据地。
> - 有攸往：有所往，指主动出击、远行（攸 = 所）
> - 建侯：建立诸侯，即建立根基
> - 屯是万物初生之卦，刚开始时别急着往外跑，先把根基立住。

**② 蒙卦：**

> 道路通达。不是我求着蒙昧之人，而是蒙昧之人来求我。
> - 童蒙：蒙昧的幼童，比喻不懂的人、需要被教导的人
> - 深层含义：学习的主动性在学生那边，老师不该追着学生教

**③ 需卦：**

> 有信心，前途光明亨通，守正则吉，利于渡过大河。
> - 需 = 等待，核心不是"马上去做"，而是"有信心地等待时机"
> - 孚：不只是信心，更接近诚心、诚信——等待时是真诚的、不焦虑的

</details>

**练习三：填空**

> "元亨利贞"四个字中：
>
> - 描述事物根基的是____
> - 描述通道顺畅的是____
> - 表示"适合去做"的是____
> - 行为建议（守正）的是____

<details>
<summary>▸ 答案</summary>

元（根基）· 亨（通达）· 利（适宜）· 贞（守正）

</details>

### 第五课小结

| 要点 | 内容 |
|:---|:---|
| 卦辞的两个零件 | 叙述句（描述处境）+ 断语（给出判断） |
| 断语光谱 | 吉 > 吝 > 悔 > 厉 > 凶 |
| 亨与贞 | 亨 = 通达；贞 = 守正（行为建议，非结果） |
| 四德 | 元（根基）→ 亨（通达）→ 利（适宜）→ 贞（守正） |
| 核心提醒 | "利贞"是有条件的指引，不是保证 |
